# Blockquotes

Blockquote components for highlighting important quotes, testimonials, and emphasized content.

## Basic Blockquote

Simple blockquote for highlighting important text:

<div className="my-6">
  <Blockquote>
    The best way to predict the future is to create it.
  </Blockquote>
</div>

```jsx
import { Blockquote } from '@openkite/kite-ui'

<Blockquote>
  The best way to predict the future is to create it.
</Blockquote>
```

## Blockquote with Attribution

Add attribution to provide context and credibility:

<div className="my-6">
  <Blockquote>
    Design is not just what it looks like and feels like. Design is how it works.
    <footer className="mt-2 text-sm text-gray-600">
      — Steve Jobs
    </footer>
  </Blockquote>
</div>

<Blockquote>
  Design is not just what it looks like and feels like. Design is how it works.
  <footer className="mt-2 text-sm text-gray-600">
    — Steve Jobs
  </footer>
</Blockquote>
```

## Warning or Important Notes

Use blockquotes to highlight important information:

<div className="my-6">
  <Blockquote className="border-l-yellow-400 bg-yellow-50">
    <strong>Important:</strong> Always test your components with screen readers and keyboard navigation 
    to ensure they meet accessibility standards.
  </Blockquote>
</div>

```

## Best Practices

- Use blockquotes to highlight important or inspiring content
- Always provide attribution when quoting others
- Keep quotes concise and impactful
- Use appropriate styling to match your design system
- Consider accessibility when styling blockquotes
- Use semantic HTML structure for better SEO
